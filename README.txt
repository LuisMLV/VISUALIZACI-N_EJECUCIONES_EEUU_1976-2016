Este repositorio consiste en un proyecto de visualización creado en Stackblitz, que consiste en la visualización de diferentes características de un conjunto de datos sobre las ejecuciones en Estados Unidos entre 1976 y 2016. Este proyecto contiene visualizaciones creadas en su mayoría con el lenguaje de programación python y sus librerías matplotlib y seaborn.

El contenido de este repositorio es el siguiente:


 - Los documentos típicos de un proyecto web y un proyecto de stackblitz: index.html, index.js, package.json, y style.css.
 - Archivo VISUALIZACIONES-PYTHON.py, archivo de python que contiene el código con el que se ha generado la mayoría de visualizaciones del proyecto
 - El presente archivo, README.txt, que describe el contenido de este proyecto.
 - executions.csv, archivo csv que contiene el conjunto de datos utilizado en el proyecto, extraído del sitio de Kaggle https://www.kaggle.com/usdpic/execution-database .
 

Enlace al proyecto de Stackblitz: https://stackblitz.com/edit/js-axqpfo .
